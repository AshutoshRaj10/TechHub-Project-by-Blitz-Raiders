# DB Analyser Bot Frontend 🎨

This directory contains the React-based frontend for **DB Analyser Bot**, a professional data intelligence bot developed by **Team Blitz Raiders**.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
npm install
```

### Running in Development
```bash
npm run dev
```

### Building for Production
```bash
npm run build
```

## 🏗️ Architecture

- **React + Vite**: For a fast development experience.
- **Tailwind CSS**: For styling and responsive design.
- **D3/Chart.js**: For data visualizations.
- **Mermaid.js**: For generating ER diagrams and flowcharts.
- **Lucide React**: For icons.

## 🔗 Connection to Backend
The frontend expects the backend to be running on `http://localhost:8000` by default. You can configure this in the `vite.config.js` or through environment variables if implemented.
